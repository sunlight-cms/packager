Installation

  1. Upload files to your web server

    Upload the contents of the cms directory to a public directory on your web server (e.g. via FTP).

  2. Setup permissions

    UNIX web servers may require you to set permissions of the uploaded files to at least 644 to allow making backups, using the file manager, uploading avatars, adding images to galleries, generating thumbnails etc.

  3. Install the database

    Open the /install path in your web browser (e.g. http://yourwebsite1/install/). Continue per instructions on screen. After the installation is complete, you will have to remove the install directory.

That's it

For more information, discussion, plugins and technical support visit the official website at sunlight-cms.cz.
